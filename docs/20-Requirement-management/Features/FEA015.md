### Testing / possible acceptance criteria

- PrestaScan Security is successfully integrated and scans the PrestaShop website periodically. (Test Case)

- Malware and known vulnerabilities are detected and listed in a report. (Test Case)

- Security Officer receives real-time alerts when vulnerabilities or threats are identified. (Test Case)

- Guidance or recommendations for fixing detected issues are provided via the tool. (Test Case)

- Users can access basic security information and results through a simple, easy-to-use interface. (Test Case)

- The scanning solution is free to use and requires minimal setup, suitable for learning purposes. (Test Case)
