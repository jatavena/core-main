### User interface mock-up

No changes to user interface

### Testing / possible acceptance criteria

- Users can set up a PrestaShop store through a simplified, guided process without needing technical knowledge.

- 24/7 technical support is available with timely responses, and users can access a knowledge base for common issues.

- Users have access to a dashboard with key analytics (sales, traffic, conversions) and can export reports in common formats.
